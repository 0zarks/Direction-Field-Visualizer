# Direction Field Visualizer (ODE Slope Fields)

An open-source Python tool for visualizing **direction fields / slope fields**
for first-order differential equations of the form

dy/dx = f(x, y)

Built for **Differential Equations students**, with:
- Color-coded slope fields
- Support for trigonometric, exponential, and root functions
- Runs entirely in **Google Colab** (no installation required)
